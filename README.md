# 打音法 固有振動数解析アプリ / Tapping (Impact-Echo) Test Analyzer

打音法（インパクトエコー法）で音を録音しながら、リアルタイムに **波形・スペクトル・スペクトログラム** を表示し、
**卓越周波数（固有振動数 f₀）** を評価できる単体HTMLアプリです。インストール不要、ブラウザだけで動作します。

🔗 **公開URL（GitHub Pages）:** https://meta-ochir.github.io/tapping-test-analyzer/

## 主な機能

- 🎙 マイク録音しながらリアルタイム解析（PC / iPhone 対応）
- 🌊 波形（時間領域）／📊 スペクトル（周波数領域）／🔥 スペクトログラム を同時表示
- 🔨 **打撃トリガー自動捕捉**：閾値を超えた瞬間を自動検出してスペクトルを固定
- 🎯 **ピーク自動検出**：固有振動数 f₀ と倍音、Q値（f/Δf）を算出
- ⏱ **卓越周波数の数値ホールド**：f₀ を設定秒数だけ静止表示して読み取りやすく
- 🎚 **ノイズ低減**：環境ノイズ学習＋スペクトル減算、時間・周波数方向の平滑化
- 📐 各グラフに**目盛り・単位**を表示
- 💾 保存：録音 WAV / スペクトル CSV / グラフ PNG / 測定条件・結果 JSON / 測定ログ CSV
  - iPhone では共有シートから「ファイルに保存」「写真に保存」

## 使い方

1. ページを開く（iPhone は **https://** のURLで開くこと。マイクにHTTPSが必須）
2. **① マイク開始**（マイク使用を許可）
3. 静かな状態で **🎚 ノイズ学習** →（必要なら）**✔ ノイズ除去**
4. **② 打撃トリガー 待受** → 対象をハンマーで一打 → f₀ が数値ホールド表示
5. **保存 / 共有** から必要なデータを出力

## 技術メモ

- Web Audio API（`AnalyserNode` + 自前の Radix-2 FFT）
- 窓関数：Hann / Hamming / Blackman / 矩形、FFTサイズ 2048〜16384 可変
- 周波数分解能 Δf = サンプリング周波数 / FFTサイズ
